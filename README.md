# Extract Archive

One-click archive extraction for macOS. Right-click any `.rar`, `.zip`, `.tar` or 20+ other formats in Finder and extract instantly.

**Website:** [yanmcruz.github.io/extract-archive](https://yanmcruz.github.io/extract-archive)

## Quick Install

**Easy (recommended):** Download the [DMG](https://github.com/yanmcruz/extract-archive/releases/latest/download/Extract.Archive.dmg), open it, drag the app to Applications, then launch it to complete setup.

**Terminal:** Download the [ZIP](https://github.com/yanmcruz/extract-archive/archive/refs/heads/main.zip), extract it, then run:
```bash
chmod +x install.sh && ./install.sh
```

## Supported Formats

RAR, ZIP, TAR, TAR.GZ, TAR.BZ2, TAR.XZ, 7Z, GZ, BZ2
